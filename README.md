# Commandes Musique

**Gp** : Lance une musique via un nom d'artiste, une musique, un lien Youtube.
```
Exemple: 1) Gp Soprano
         2) Gp Ninja
         3) Gp https://www.youtube.com/watch?v=8hdBIi5c8ZE
```
